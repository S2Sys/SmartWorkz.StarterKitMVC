PERMISSION MATRIX
===============================================
Admin    → Full CRUD access (all resources)
Manager  → Read/Update products & orders + reports
Staff    → Read-only access
Customer → Role-based access

TROUBLESHOOTING
===============================================
401 Unauthorized → Re-login, copy new token
403 Forbidden    → User lacks permissions
400 Bad Request  → Check request format

HTTP STATUS CODES
===============================================
200 OK              → Success
201 Created         → Resource created
204 No Content      → Success (no response body)
400 Bad Request     → Validation error
401 Unauthorized    → Invalid/expired token
403 Forbidden       → Insufficient permissions
404 Not Found       → Resource not found
500 Internal Server → Server error

COMMON REQUESTS
===============================================

Login:
  POST /api/auth/login
  Body: {"email":"admin@smartworkz.test","password":"TestPassword123!","tenantId":"DEFAULT"}

List Products (with pagination):
  GET /api/DEFAULT/products?page=1&pageSize=10
  Header: Authorization: Bearer {token}

Search Products:
  GET /api/DEFAULT/products/search?q=electronics&page=1&pageSize=10
  Header: Authorization: Bearer {token}

Create Product (Admin only):
  POST /api/DEFAULT/products
  Body: {"name":"Product Name","sku":"SKU123","price":99.99,"categoryId":1}
  Header: Authorization: Bearer {token}

Change Password (Authenticated):
  POST /api/auth/change-password
  Body: {"currentPassword":"TestPassword123!","newPassword":"NewPassword123!"}
  Header: Authorization: Bearer {token}

Refresh Token:
  POST /api/auth/refresh
  Body: {"accessToken":"{token}","refreshToken":"{refresh_token}"}
